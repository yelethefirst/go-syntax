# Exercises: Function Closures

### EXERCISE 1: The Multiplier
Create a closure factory called `makeMultiplier` that returns a function. The returned function should multiply its input by a fixed factor set when the factory was called.

### EXERCISE 2: Separate States
Create two multipliers (one for `3` and one for `10`). Test them to ensure they maintain their own separate internal states.

## How to Run the Solution
```bash
go run solution/main.go
```
