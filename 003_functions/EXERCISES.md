# Exercises: Functions

### EXERCISE 1: Multiply
Create a function called `multiply` that takes two `int` parameters and returns their product.

### EXERCISE 2: Swap
Create a function called `swap` that takes two `string` parameters and returns them in reverse order (e.g., `swap("hello", "world")` returns `"world", "hello"`).

---

## How to Run the Solution
```bash
go run solution/main.go
```
