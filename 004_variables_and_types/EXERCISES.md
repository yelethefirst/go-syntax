# Exercises: Variables and Types

### EXERCISE 1: Declarations
Declare a variable `age` using the `var` keyword and another variable `name` using the short assignment operator `:=`.

### EXERCISE 2: Type Conversion
Try to add an `int` and a `float64` together. What error do you get? Fix it by converting the `int` to a `float64`.
