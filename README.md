# Go Syntax Learning Repository

Welcome to the Go Syntax Learning Repository! This repository is designed to help you learn and practice Go (Golang) syntax through structured examples and exercises.

## Repository Structure

- **[001_foundation/](001_foundation/):** Why Go? Philosophy, Pros/Cons, and Comparisons.
- **[002_packages/](002_packages/):** How to organize code and use the Standard Library.
- **[003_functions/](003_functions/):** How to write reusable tasks and return values.
- **[004_variables_and_types/](004_variables_and_types/):** Understanding 'var', ':=', and basic data types.
- **[005_flow_control/](005_flow_control/):** Logic and control statements including `for`, `if/else`, `switch`, and `defer`.
- **[006_pointers/](006_pointers/):** Memory addresses, and the `&` and `*` operators.
- **[007_structs/](007_structs/):** Grouping data together into custom types.
- **[008_arrays_and_slices/](008_arrays_and_slices/):** Working with fixed and dynamic collections.
- **[009_range/](009_range/):** Iterating through collections with ease.
- **[010_maps/](010_maps/):** Deep dive into key-value storage.
- **[011_function_values/](011_function_values/):** Functions as values and passing them as arguments.
- **[012_function_closures/](012_function_closures/):** Understanding closures and private state.
- **[013_methods_and_interfaces/](013_methods_and_interfaces/):** Defining methods, receivers, and implicit interfaces.
- **[014_generics/](014_generics/):** Write code that works with any type.
- **[015_concurrency/](015_concurrency/):** Goroutines, channels, and synchronization primitives.

## Getting Started

1.  Make sure you have [Go installed](https://golang.org/doc/install).
2.  Clone this repository.
3.  Each directory contains examples and small exercises to help you understand specific Go concepts.

## How to Run Examples

Within each directory, you can run the Go files using:

```bash
go run main.go
```

Happy coding!
