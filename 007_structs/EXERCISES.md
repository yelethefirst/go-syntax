# Exercises: Structs

### EXERCISE 1: Define a Car
Define a struct called `Car` with fields for `Make` (string), `Model` (string), and `Year` (int).

### EXERCISE 2: Park the Car
Create an instance of your `Car` struct and print its details using `fmt.Println`.
