# Exercises: Function Values

### EXERCISE 1: Simple Addition
Create a function value called `add` that takes two integers and returns their sum.

### EXERCISE 2: The Operator
Create a function `apply` that takes an integer and a function (that takes/returns an int), and returns the result of applying that function to the integer.

## How to Run the Solution
```bash
go run solution/main.go
```
