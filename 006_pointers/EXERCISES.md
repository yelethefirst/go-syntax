# Exercises: Pointers

### EXERCISE 1: Pointing to Data
Create a function that takes a pointer to an `int` and doubles the value at that address.

### EXERCISE 2: Testing the Pointer
In the `main` function, create an integer, call your doubling function with its address, and verify the value changed.

---

## How to Run the Solution
```bash
go run solution/main.go
```
