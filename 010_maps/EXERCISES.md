# Exercises: Maps

### EXERCISE 1: City Population
Create a map that stores the population of at least 3 cities (e.g., `map[string]int`).

### EXERCISE 2: Checking Existence
Check if "Tokyo" exists in your map using the "ok" idiom (`val, ok := map[key]`) and print the result.
