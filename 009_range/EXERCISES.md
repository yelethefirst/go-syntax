# Exercises: Range

### EXERCISE 1: The Sum
Given a slice of integers `[]int{2, 4, 6, 8, 10}`, use `range` to calculate and print their sum.

### EXERCISE 2: The Index
Use `range` to print only the indices (0, 1, 2, ...) of the slice without printing the values.

## How to Run the Solution
```bash
go run solution/main.go
```
